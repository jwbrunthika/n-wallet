# N Wallet Admin Web (Flutter + GetX)

## Run
```bash
flutter pub get
flutter run -d chrome --dart-define=API_BASE_URL=http://localhost:8080/api/v1
```

## Implemented Pages
- Admin login
- Dashboard
- Halls CRUD
- Beacons CRUD
- Modules CRUD
- Timetable CSV import
- Sessions list
- Students list + enrollment images + reset enrollment
- Attendance logs + CSV export preview
- Settings (face/beacon thresholds)
